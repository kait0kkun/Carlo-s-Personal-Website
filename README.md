# Carlo Bunayos — Personal Portfolio

A professional, modern, and responsive personal portfolio website built for Carlo Bunayos, Vice President of Sales at Lampara Publishing House, Inc. 

## Features
- **Modern UI/UX**: Premium aesthetic featuring a Navy (#0A1628), Gold (#C9A84C), and White color palette.
- **Scroll Reveal Animations**: Smooth entry animations as users scroll down the page, enhancing engagement.
- **Responsive Design**: Fully optimized for mobile phones, tablets, and large desktop screens without compromising layout integrity.
- **Dynamic Projects & Testimonials**: Organized project showcases and an auto-rotating testimonial carousel.
- **SEO Optimized**: Built with semantic HTML5 and clean architectural hierarchy.

## Technologies Used
- HTML5
- CSS3 (Custom Variables, Flexbox, CSS Grid)
- Vanilla JavaScript (DOM manipulation, scroll handling)

## Getting Started

To view the website locally, you can use any standard static web server. 

If you have Node.js installed, you can easily serve the files from your terminal:
```bash
npx serve .
```
Then, open your browser and navigate to the localhost port provided (usually `http://localhost:3000`).

## Deployment
This static website is fully ready to be deployed to platforms like GitHub Pages, Vercel, or Netlify with zero build configurations required. Simply link the repository or upload the files to your host of choice.
